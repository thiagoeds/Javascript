Nesta aula vamos falar sobre o querySelector, seleciona a primeira opção satisfatoria, exemplo: document.querySelector('.lista');.

Mesmo tendo duas class="lista", ele vai pegar apenas a lista não ordenado, porque é a primeira opção dele.
ul class="lista" ul

Temos também o querySelectorAll, diferente do primeiro que falamos, este vai pegar todos os elementos com a mesma class ou id presentes no meu codigo html, e retorna uma array para que assim eu possa selecionar a opção desejada.
Vejamos o exemplo:

document.querySelectorAll('.lista'), no console ele vai retornar todas as opções com a mesma identificação e array do elemento.

0: ul.lista
1: ol.lista
