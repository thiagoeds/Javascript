Vamos continuar abordando as ferramentas de dimensões no console.

Iniamos realizando a busca pelo class que quero saber as informações com a informação a seguir: document.querySelector('.texto'), para descobrir o tamanho do scroll da pagina vou repetir a operação com a barra de scroll ou barra de rolagem no meio ou no fim para saber o valor exato document.querySelector('.texto').scrollTop, no caso o texto acima tem 80px.

Vamos fazer agora um botão para quando eu queira voltar para o top da pagina, basta que eu clique nele, então vamos?

Primeiro passo é criar um botão clicavel, apos no javascript inserimos o seguinte codigo:

O nosso subirtela é a nossa identificação para o onclick.

function subirtela() {
window.scrollTo({
top: 0,
behavior: "smooth",
});
}

Podemos criar também outra função, desta vez vamos fazer também no javascript, na qual quando estivermos no topo da pagina o botão não ficara visivel e a cada 1 segundo, será realizada uma nova verificação, caso a barra de scroll saia do topo da página o botão vai aparecer novamente, possibilitando assim ser clicado.

Existe duas possibilidades de criar esta função, a primeira delas é por tempo, porém esta não é a mais efitiva, pois ele fica executando a função de segundo em segundo, veja no exemplo a seguir:

function decidirBotaoScroll() {
if(window.scrollY "Y" na frente de scroll, significa o eixo de rolagem, lembrando que também temos o eixo "X" === 0) {
//ocultar botão
document.querySelector('.scrollbutton').style.display = 'none';
}else{
//mostrar botão
document.querySelector('.scrollbutton').style.display = '';
}
}

setInterval(decidirBotaoScroll, 1000 1000 representa aqui 1 segundo );

Temos também a outra opção, esta mais avançada, porém mais efetiva; Neste exemplo vamos criar um evento que vai monitorar o scroll da tela.

window.addEventListener('scroll', decidirBotaoScroll);
