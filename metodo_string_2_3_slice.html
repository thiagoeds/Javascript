O metodo slice

slice() extrai uma parte de uma string e retorna a parte extraída em uma nova string.
Então vamos lá!

1º Exemplo:

O método leva 2 parâmetros: a posição inicial e a posição final (final não incluído).
Este exemplo corta uma parte de uma string da posição 7 à posição 12 (13-1):

Para isso vamos passar os sequintes parametros:

let frutas = "Maça, Banana, Kiwi";

document.getElementById("teste").innerHTML = frutas.slice(6,12);
// Retornar Banana

Quero que retorne na frase a seguir a palavra Banana.

3º Exemplo:

Com o Slice é possivel também selecionar a partir de numeros negativos, a diferença é que sera selecionado o conteudo do fim para o inicio, ou seja, a seleção começa do inverso.

Veja o exemplo usando a memsma frase.

let frase = 'Bonieky Lacerda Leal';

document.getElementById("teste3").innerHTML = frase.slice(-4);
