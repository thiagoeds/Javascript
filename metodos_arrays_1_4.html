O metodo Arrays

Usando o metodo "toString", ele separa cada palavra separada por virgula, se transforma em uma Array.

let lista = ['Ovo', 'Farinha', 'Corante', 'Massa'];

let exemplo = lista.toString();

document.getElementById("exemplo").innerHTML = exemplo;

Usando o metodo "join(" - ")", ele separa cada palavra separada por virgula, se transforma em uma Array.

let lista1 = ['Ovo', 'Farinha', 'Corante', 'Massa'];

let exemplo2 = lista1.join(" - ");

document.getElementById("exemplo2").innerHTML = exemplo2;

Usando o metodo "indexOf('Corante')", ele tras a informação da posição do elemento dentro da Array, caso ele não encontre o elemento, ele vai retornar "-1".

let lista3 = ['Ovo', 'Farinha', 'Corante', 'Massa'];

let exemplo3 = lista3.indexOf('Corante');

document.getElementById("exemplo3").innerHTML = exemplo3;

Usando o metodo "pop", ele simplismente remove o ultimo item dentro da Array.

let lista4 = ['Ovo', 'Farinha', 'Corante', 'Massa'];

lista4.pop();

let exemplo4 = lista4;

document.getElementById("exemplo4").innerHTML = exemplo4;

Usando o metodo "shift", ele simplismente remove o primeiro item dentro da Array.

let lista5 = ['Ovo', 'Farinha', 'Corante', 'Massa'];

lista5.shift();

let exemplo5 = lista5;

document.getElementById("exemplo5").innerHTML = exemplo5;
