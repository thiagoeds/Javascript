O metodo Substring

substring() é semelhante a slice().
A diferença é que substring() não aceita índices negativos.
Então vamos lá!

1º Exemplo:

O método leva 2 parâmetros: a posição inicial e a posição final (final não incluído).
Este exemplo corta uma parte de uma string da posição 7 à posição 12 (13-1):

Para isso vamos passar os sequintes parametros:

let frutas = "Maça, Banana, Kiwi";

document.getElementById("teste").innerHTML = frutas.substring(6,12);
// Retornar Banana

Quero que retorne na frase a seguir a palavra Banana.
