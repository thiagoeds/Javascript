Existe diferentes formas de se fazer uma Array, as que se segue são mais avançadas:

Uma unica Array com todas as opções dentro dela e até mesmo a função.

let carros = ['Palio', 'Uno', 'Corolla', 'Ferrari', function() {
console.log('testando 1, 2, 3...')
}]

Dando o seguinte comando (carros) no console, vai retornar a seguinte informação:

(5) ["Palio", "Uno", "Corolla", "Ferrari", ƒ]
0: "Palio"
1: "Uno"
2: "Corolla"
3: "Ferrari"
4: ƒ ()
length: 5
__proto__: Array(0)

Array dentro de Array.

let ingredientes = [
['Uva', 'Pera', 'Maça'],
['Arroz', 'Macarrão', 'Feijão']
]

Dando o seguinte comando (console.log(ingredientes);) no console, vai retornar a seguinte informação:
